This repository contains end-to-end data analytics projects developed using Power BI.

Each project focuses on transforming raw data into meaningful insights through data profiling, modeling, DAX measures, and interactive dashboards.

Data Analytics Workflow:

Data Gathering 

• Collected Sales and UPI transaction datasets 

• Used structured files (Excel / CSV) 

• Identified key fields for analysis

Data Transformation 

• Cleaned data using Power Query 

• Removed duplicates and null values 

• Standardized date and numeric formats 

• Created calculated columns (Net Sales, Balance, Categories) 

• Performed data profiling

Data Modeling 

• Created relationships between tables 

• Optimized model for performance 

• Built DAX measures (Sales, Profit, Quantity, Orders, Discount) 

• Ensured proper filter flow

Reporting & Visualization 

• Built interactive dashboards 

• Used slicers for dynamic filtering 

• Created time-based trend analysis 

• Designed matrix and comparison visuals 

• Applied conditional formatting 

• Used bookmarks for navigation
